1. Giải thích các câu lệnh:
    git commit --amend: Mở editor cho bạn sửa commit message và nội dung commit
    git commit --amend -m"<message>": Đổi commit trực tiếp, không mở editor
    git restore --staged <file>: đưa file từ vùng staging về directory 
    git reset HEAD~1.: đưa tất cả từ vùng repository về working directory
